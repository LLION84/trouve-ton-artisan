# Trouve ton artisan 🔨

Plateforme dédiée aux artisans de la région Auvergne-Rhône-Alpes.

## Prérequis

- Node.js v18+
- npm v9+

## Installation

```bash
git clone https://github.com/LLION84/trouve-ton-artisan.git
cd trouve-ton-artisan
npm install
```

## Lancement

```bash
npm start
```

Ouvre [http://localhost:3000](http://localhost:3000) dans ton navigateur.

## Build production

```bash
npm run build
```

## Technologies utilisées

- React 18
- React Router DOM
- Bootstrap 5
- Sass
- Bootstrap Icons

## Fonctionnalités

- Recherche d'artisans par nom, spécialité et ville
- Navigation par catégorie (Bâtiment, Services, Fabrication, Alimentation)
- Fiche détaillée par artisan
- Formulaire de contact
- Site responsive — mobile, tablette, desktop
- Page 404
